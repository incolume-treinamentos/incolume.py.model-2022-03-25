# Modelo de projeto Python #

---

## Objetivos ##
Base com estrutura inicial para começar um projeto Python.

## Descrição ##
Estrutura base genérica para iniciar um projeto Python.

## Escopo
- Sistema operacional Linux preferencialmente;
- git
- Python 3.6.2+
- poetry

### Ferramentas aplicadas
- black
- coverage
- isort
- mypy
- pydocstyle
- pylama
- pylint
- pytest
- pytest-cov
- safety
- sphinx
